# 🐤 Flappy Bird Game (C++)

## 📌 Overview

Flappy Bird is a console-based game developed in C++ that simulates the classic arcade-style gameplay. The player controls a bird that must pass through obstacles without colliding. The game focuses on real-time input handling, score tracking, and game loop mechanics.

---

## 🎮 Features

* Real-time keyboard input handling
* Dynamic obstacle generation
* Score tracking system
* Collision detection logic
* Game loop implementation
* Increasing difficulty over time

---

## 🛠️ Technologies Used

* C++
* Console-based UI
* OOP Concepts
* File Handling (if used for score saving)

---

## 💡 Concepts Implemented

* Loops and Conditional Statements
* Functions and Modular Programming
* Object-Oriented Programming (basic level)
* Game logic design
* Event handling

---

## 🚀 How to Run

1. Open project in Dev-C++ or Code::Blocks
2. Compile the program
3. Run the executable file
4. Use keyboard keys to control the bird

---

## 📌 Project Purpose

This project was developed to understand game logic, real-time user interaction, and structured programming in C++.
